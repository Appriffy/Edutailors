The steps are so simple, all it requires is latest version of python and django

1.Python Version = 3.8.1
2.Recommend to use Chrome(incognito mode)/Mozilla
3.Runs on Linux/Windows OS 

Steps to run the project

Extract proj07012020.rar 

$ cd proj07012020
$ cd appriffy_demo
$ pip3 install -f requirement.txt
or
$ pip3 install asgiref django django-cors-headers djangorestframework Pillow pytz sqlparse
$ python manage.py runserver

Open the browser with following entry.

http://localhost:8000/

click the votes button and check the desired output

Note: Please don't change the port number, This demo application was developed with latest version of tools, there may be some bugs in the latest version of tools.
